# 04. マーケティング戦略・企画（Marketing Strategy）

このファイルでは、市場環境の分析からターゲット選定、そして具体的な顧客体験の設計までを一貫して行うためのプロンプトをまとめています。
「分析」→「戦略」→「顧客理解」の順に進めることで、精度の高い企画立案が可能です。

---

## 1. 環境分析（Macro & Micro Analysis）
市場の全体像を把握し、自社の立ち位置を明確にします。

### 🌍 PEST・3C・5F分析（市場環境の把握）
Webブラウジング機能（検索）を活用し、最新の市場動向を多角的に分析します。

```text
# 命令書
あなたは世界トップクラスの戦略コンサルティングファームのパートナーです。
以下の {業界・テーマ} について、Webブラウジング機能を活用して最新情報を収集し、多角的な分析を行ってください。

# 分析フレームワーク
1. PEST分析（政治、経済、社会、技術）：マクロ環境の動向を具体的に列挙すること。
2. 5F分析（競合、新規参入、代替品など）：業界の収益構造と脅威を明らかにすること。
3. 3C分析（市場・顧客、競合、自社）：主要な競合プレイヤーと、成功要因（KSF）を特定すること。

# 制約条件
- 情報は必ず直近1年以内の信頼できるソース（公的機関、業界レポート等）に基づくこと。
- 分析結果に基づき、この業界における「最大のチャンス」と「最大のリスク」を結論付けること。

# 業界・テーマ
（例：日本の電気自動車（EV）市場）
```

### ⚔️ SWOT・クロス分析（戦略の方向性）
環境分析の結果を踏まえ、自社が取るべき戦略オプションを導き出します。

```text
# 命令書
あなたは論理的思考に優れたマーケターです。
先ほどの環境分析の結果と、以下の {自社の特徴} を踏まえ、SWOT分析を行ってください。

# ステップ
1. SWOT表の作成：強み、弱み、機会、脅威を整理する。
2. クロスSWOT分析：
   - 強み × 機会（積極攻勢）：最大のチャンスをどう活かすか？
   - 弱み × 機会（改善）：チャンスを逃さないために何を克服すべきか？
   - 強み × 脅威（差別化）：強みを活かして脅威をどう回避するか？
3. 戦略案の提示：上記から導かれる、優先すべき戦略オプションを3つ提案する。

# 自社の特徴
（ここに自社の強みやリソース情報を貼り付ける）
```

## 2. ターゲット選定（STP Analysis）
「誰に」「どのような価値」を届けるかを定義します。

### 🎯 セグメンテーション・ターゲティング・ポジショニング
市場を細分化し、狙うべきポジションを明確にします。

```text
# 命令書
あなたは**鋭い洞察力を持つ**マーケティング・ディレクターです。
これまでの分析を元に、STP分析を行ってください。

# 出力項目
1. セグメンテーション（軸の決定）：
   - 市場を分けるための「独自の切り口（軸）」を2つ提案し、4象限のマトリクスを作成してください。
   - ※ありきたりな「年齢・性別」だけでなく、「価値観」や「利用シーン」などの軸も検討すること。
2. ターゲティング（標的の決定）：
   - 4象限のうち、当社が狙うべきセグメントはどこか？その理由は？
3. ポジショニング（差別化）：
   - 競合と比較した際、当社が「No.1」または「Only 1」になれるポジションはどこか？

# ポジショニングマップの軸案
縦軸： {例：価格の高さ vs 安さ}
横軸： {例：機能の豊富さ vs 操作のシンプルさ}
```

## 3. 顧客理解（Persona & Journey）
ターゲットユーザーを深く理解し、具体的な体験を設計します。

### 👤 ペルソナ定義（詳細プロフィール）
ターゲット像を、実在する人物のように詳細に描き出します。

```text
# 命令書
あなたは人間心理に精通した小説家兼マーケターです。
選定したターゲットセグメントを象徴する「ペルソナ（架空の顧客像）」を1名、詳細に設定してください。

# プロフィール項目
- 名前、年齢、職業、家族構成
- 性格・価値観（何を大切に生きているか）
- 現状の不満・悩み（解決したい深い悩み）
- 口癖（よく言うセリフ）
- 情報収集源（よく見るSNS、メディア）

※出力後、このペルソナの顔画像を生成するための「画像生成プロンプト」も作成してください。
```

### 🗺️ カスタマージャーニーマップ（体験設計）
ペルソナがサービスを知り、利用し、ファンになるまでの過程を可視化します。

```text
# 命令書
あなたはUX（ユーザー体験）デザイナーです。
作成したペルソナが、当社のサービスを認知してから購入（契約）、そして継続利用に至るまでの「カスタマージャーニー」を表形式で作成してください。

# カラム構成（表の列）
1. フェーズ（認知 → 興味 → 比較検討 → 購入 → 利用 → 推奨）
2. 顧客の行動（具体的なアクション）
3. タッチポイント（スマホ検索、SNS、店舗など）
4. 思考・感情（「おっ、いいな」「本当に大丈夫かな？」など）
5. 潜在ニーズ・インサイト（本人も気づいていない本音）
6. 当社のアプローチ（このタイミングで打つべき施策）

# 出力形式
Markdownの表形式
```
