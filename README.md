# ai-training-prompts

AI学習用プロンプト集 - 様々なタスクに対応した効果的なプロンプトのコレクション

### 概要

このリポジトリは、AI（ChatGPT、Claude、Geminiなど）との対話を効率化し、より良い結果を得るためのプロンプト集です。プログラミング、文章作成、分析、創作など、様々な用途に特化したプロンプトを提供しています。

目的に応じたカテゴリー別に整理されており、すぐに使える実例も豊富に含まれています。初心者から上級者まで、幅広いユーザーが活用できる内容となっています。

### 構成

- `00_Basic_Techniques.md` - 生成AIの基本的な使い方とテクニック
- `01_Writing_Operations.md` - 文章作成・校正・要約のプロンプト
- `02_Ideation_Presentation.md` - アイデア発想とプレゼン資料作成のプロンプト
- `03_Data_Analysis.md` - データ分析・可視化のプロンプト
- `04_Marketing_ContentCreation.md` - マーケティング・コンテンツ作成のプロンプト
- `05_Sales_CustomerSuccess.md` - 営業・カスタマーサクセスのプロンプト
- `06_Engineering_IT.md` - エンジニアリング・IT関連のプロンプト
